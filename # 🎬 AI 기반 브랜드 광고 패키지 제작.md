생성형 AI(텍스트·이미지·비디오·오디오)를 활용해
**10초 이내 브랜드 광고 영상**과 **스토리보드 기획 문서**를 제작하는 과제입니다.

## 1. 프로젝트 개요
> 멀티모달 생성 AI 도구를 조합하여 하나의 완성된 브랜드 광고를 기획·제작합니다.
> 도구 선택 근거와 프롬프트 설계 과정을 문서로 남기는 것이 핵심입니다.

---

## 2. 최종 산출물

### 📄 스토리보드 문서 (PDF 1개)
- 브랜드 아이덴티티
- 캠페인 목표 / 핵심 메시지
- 씬 구성 (아래 필수 필드)
- 사용 도구 목록

**씬별 필수 필드 (모든 씬 공통)**

| 필드 | 설명 |
|------|------|
| 씬 번호 / 길이(초) | 씬 구분과 재생 시간 |
| 목표 메시지 | 한 문장 |
| 화면 구성 | 구도 / 피사체 / 배경 / 텍스트 유무 |
| 내레이션·카피 | 음성 또는 화면 텍스트 |
| 사용 도구 | 이미지/비디오/오디오 + 사용 목적 |
| 프롬프트 | 원문 + 출력 결과 요약(한 줄) |
| 결과 파일 | 파일명 또는 링크 (규칙 자유, 일관성 유지) |

> 💡 **권장**: 최소 1개 씬은 “프롬프트 수정 전/후”와 수정 이유를 기록

### 🎞 광고 영상 (MP4 1개)
- 길이: **10초 이내**
- 생성형 AI 결과물을 **주된 소스**로 사용
- **시각 요소(이미지/영상) + 청각 요소(음성/효과음/BGM 중 1개 이상)** 필수 포함

## 3. 학습 목표
과제를 마치면 다음을 스스로 설명할 수 있어야 합니다.
- [ ] 텍스트/이미지/비디오/오디오 도구의 역할과 강·약점 비교
- [ ] 기획 의도에 맞춘 프롬프트 설계·수정 과정
- [ ] 여러 도구 결과 비교 및 도구 선택 근거
- [ ] 미디어 소스를 해상도/비율/톤앤매너로 정리해 하나로 통합하는 흐름

## 4. 필수 요구사항

### ① 기획 정의
- 브랜드(가상/실제) 1개 선정 → **타겟 / 톤앤매너 / USP** 포함 아이덴티티 정의
- 광고 목적(인지/전환/리텐션 등) + **핵심 메시지 1개**를 한 문장으로 정의

### ② 스토리보드 문서화
- 씬 단위 작성, 필수 필드 누락 없이 기입
- 최소 1개 씬에서 프롬프트 **수정 전/후** 비교 (무엇을·왜)

### ③ 멀티모달 도구 사용
- 이미지 생성 AI **1종 이상**
- 비디오 생성/변환 AI **1종 이상**
- 오디오 생성 AI **1종 이상**
- 각 도구의 **선택 이유**를 스토리보드에 기록

### ④ 서사 / 메시지 (완성도)
아래 중 **하나 이상** 충족:
- 기승전결 (문제 제시 → 전환 → 해결/제안)
- 명확한 메시지 전달 (누구에게 어떤 가치를 주는지)

> 마지막 3~5초에 **로고 / 브랜드명 / 슬로건 / CTA 중 1개 이상** 포함

## 5. 영상 인코딩 스펙

| 항목 | 권장 스펙 | 최소 허용 |
|------|-----------|-----------|
| 해상도 | 1080p | 720p 이상 |
| 프레임레이트 | 24~30 fps | - |
| 비디오 코덱 | H.264 | - |
| 오디오 코덱 | AAC | - |

> 최대 스펙은 필수 아님. 저사양 환경은 720p 이상이면 허용.

## 6. 보너스 과제 (선택)
- **보너스 1 – 립싱크**: 인물 발화 장면 1개 이상, 입모양·대사 자연스럽게 맞추기
- **보너스 2 – 다른 도구 재제작**: 이미지/비디오 파트를 다른 도구로 동일 씬 1~2개 재제작
- **보너스 3 – 비율 버전**: 동일 스토리보드로 화면 비율 2종 이상
  (예: 16:9 유튜브 / 9:16 숏폼 1080×1920 / 1:1 피드)

## 8. 제출 방법 & 체크리스트

**제출물**
1. `storyboard.pdf`
2. `ad_final.mp4` (10초 이내)

**제출 전 체크리스트**
- [ ] 브랜드 아이덴티티 + 핵심 메시지 1문장
- [ ] 씬별 필수 필드 모두 작성
- [ ] 프롬프트 수정 전/후 1개 이상 기록
- [ ] 이미지·비디오·오디오 AI 각 1종 이상 사용
- [ ] 영상에 시각+청각 요소 포함
- [ ] 마지막 3~5초 브랜드 인지 장치 포함
- [ ] 인코딩 스펙(720p 이상) 충족
